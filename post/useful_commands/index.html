Shut Down and Turn on Local Backup

shut down:

$ sudo tmutil disablelocal

trun on:

$ sudo tmutil enablelocal

Modify Launchpad Icon Size

$ defaults write com.apple.dock springboard-rows -int 7 # change the size by modifying the number
$ defaults write com.apple.dock springboard-columns -int 6 # change the size by modifying the number
$ killall Dock

Reset:

$ defaults write com.apple.dock ResetLaunchPad -bool TRUE
$ killall Dock

Change Source ‘.bash_profile’ to Source ‘.profile’

$ cat .bash_profile >> .profile
$ rm .bash_profile (optional)
$ echo "source ~/.profile" >> .bash_profile

Open Port 22

Note: most information is borrowed from [How do I open port 22 in OS X 10.6.7].
Handling the problem of:

$ ssh localhost
ssh: connect to host localhost port 22: Connection refused

The above problem happens since the remote login is closed in Mac OS X

$ sudo systemsetup -getremotelogin
Remote Login: off

Here is the solution:

$ sudo systemsetup -setremotelogin on
$ ssh localhost
Last login: ......
$ _

Uninstall MySQL using Terminal Completely

$ cd ~/
$ sudo rm /usr/local/mysql
$ sudo rm -rf /usr/local/var/mysql
$ sudo rm -rf /usr/local/mysql*
$ sudo rm -rf /Library/StartupItems/MySQLCOM
$ sudo rm -rf /Library/PreferencePanes/My*

Then open hostconfig:

$ vim /etc/hostconfig

and removed the line: MYSQLCOM=-YES-rm -rf ~/Library/PreferencePanes/My*, after that, removing following:

$ sudo rm -rf /Library/Receipts/mysql*
$ sudo rm -rf /Library/Receipts/MySQL*
$ sudo rm -rf /var/db/receipts/com.mysql.*
